# Jeu du plus ou moins
Vous devez trouver un nombre généré aléatoirement entre 1 et 1000.  
Le script vous dira si le nombre que vous avez rentré est supérieur ou inférieur au nombre généré.  
Vous avez autant de coups que vous le souhaitez.  
Pour lancer le script, placez vous dans dans l'invite de commande dans le dossier ou vous l'avez téléchargé et faites la commande suivante :  
`./jeu.sh`
